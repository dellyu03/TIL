# 변수

## 변수?

- 값을 저장하는 공간

```
변수는 메인 메모리에 만들어지며,
변수에 저장된 값은 바뀔 수 있다.
```

- 변수를 사용하는 이유
  - 사용자에게서 받는 데이터를 저장하기 위해
  - 직접 값을 넣는 것보다 유연하게 프로그램을 작성할 수 있음

#### ex) 사각형의 면적을 구하는 프로그램 작성

**변수를 사용하지 않는 코드**

```
area = 100 * 200;
```

```
width = 100;
height = 200;
area = width * height;
```

변수를 사용하는 코드가 더 길지만 변수 width와 height에 저장된 값을 변경하면 여러가지 사각형의 넓이를 계산할 수 있다.
