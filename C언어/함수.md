# 함수

## 함수

- 특정 작업을 수행하는 명령어들의 모음
- 서로 구별되는 이름을 가지고 있음
- 함수는 입력을 받을 수 있고 결과를 반환 할 수 있음

## 함수를 쓰면 좋은점

- 프로그램의 가독성이 좋아짐
- 유지관리가 쉬워짐
- 다른 프로그램에서도 재사용 할 수 있음

## 함수의 종류

### 라이브러리 함수

- 컴파일러에서 지원 되는 함수 ex) printf()

### 사용자 정의 함수

- 개발자가 직접 만든 함수

## 함수 정의

- 함수의 형태
  반환형 함수 이름(매개 변수(없어도 됨))
  {
  수행할 작업
  }

```C
void print_stars()
{
    for(int i = 0; i<30; i++)
        printf("*");
}

```

### 반환형

- 함수가 처리를 종료한 후에 반환하는 데이터의 유형  
  (예시의 함수는 반환값이 없어 반환형이 void)

- **매개변수 :** 함수의
- **인수 :** 호출될때 매개 변수에 전달 되는 값
