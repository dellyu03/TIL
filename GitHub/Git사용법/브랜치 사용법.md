# 브랜치 사용법

## [브랜치란](/GitHub/git%20%EA%B0%9C%EB%85%90/%EB%B8%8C%EB%9E%9C%EC%B9%98.md)

## 브랜치 확인 하기

현재 저장소의 브랜치를 표시함

```
git branch
```

확인 후 `q`를 눌러 터미널로 돌아옴

## 브랜치 생성

- 원하는 이름의 브랜치를 생성하는법

```
git branch branch-name
```

branch-name에 원하는 브랜치 이름을 입력

#### ex)

```
git branch mybranch
```

mybranch라는 이름의 브랜치 생성

## 브랜치 삭제

- 필요 없거나 잘못 만든 브랜치 삭제

```
git branch -d branch-name
```
