# Github 저장소와 로컬 파일 연동

### 1. git 설치

```
brew install git
```

### 2. 파일 생성

- 작업 파일을 저장할 파일을 생성

### 3. git 저장소 생성

- 터미널을 켜고 생성한 파일 디렉터리로 이동 후 아래 명령어로 저장소 생성

```
git init
```

## git clone

## 깃허브 저장소와 로컬 파일 연동

1.git 초기화

```
git init
```

2. 저장소 연결

```
git remote add branch "repository URL"   //연동할 저장소의 URL
```

## 원격 저장소 변경하는 법

```
git remote set-url branch new-repository
```
